# Code of Conduct

All contributors are expected to adhere to this code of conduct.

## Our Pledge
We pledge to make participation in this project a harassment-free experience.

## Expected Behavior
- Be respectful and constructive
- Use welcoming language
- Assume good faith

## Unacceptable Behavior
- Harassment, hate speech, doxxing
- Disrespectful or derogatory comments

## Enforcement
Report violations by opening an issue with `[coc]` prefix. Maintainers will review and act.
